Styleguide

Headers

Our styleguides do not rely on using the header tags unless there is no other option, such as editors using WYSIWIGs on blog posts. All header tags should be styled with a class so that we are not limited to one particular style to one particular heading level. This means we can use a particular style on a section that is not as important, but may share a stylistic element.

Note: For consistency of design, all of these styles below are actually on p tags.
